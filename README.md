# About BOXFC

![Static Badge](https://img.shields.io/badge/boxfc-started_project-magneta?style=for-the-badge)

A frontend language that's lightning quick, loads html on request, extremely small bundles. ⚡(More on this later)

# Installing it

```bash
pnpm add @boxfc/core
```

The current version of this package is built with PNPM. PNPM is recommended pacakage manager. Later support will be added for other package managers. (This is to avoid bugs)
